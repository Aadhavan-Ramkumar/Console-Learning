Console Learning

Visual Studio Code is a coding platform which allows you to create more websites. To set it up you need to download the VSCodeSetup and follow the instructions given.

Console Screens are windows where you run programs. It is used because you need it you run programs. To see the console press Ctrl+Shift+I and select Console. You can see the console there. If you want to print anything on the console you have to either type anything in it or code it in your HTML file.
